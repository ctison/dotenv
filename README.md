# Dotenv CLI written in Go

## Usage

```sh
NAME:
   dotenv - Inject envfiles into the env and start the command

USAGE:
   dotenv [-p PATH] COMMAND [ARGS ...]

GLOBAL OPTIONS:
   -e value    Inject .env file(s) into the environment
   -i          Ignore inherited environment (default: false)
   --help, -h  show help (default: false)
```
